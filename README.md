# Beauty Booking — MVP 1.0
 
Página web de reservas para negocios de belleza que aún gestionan sus citas por WhatsApp.
 
---
 
## El problema
 
Cuando un negocio de belleza empieza a crecer, gestionar citas por WhatsApp se vuelve un caos: mensajes perdidos, horarios cruzados y tiempo perdido respondiendo lo mismo todos los días.
 
## Para quién es
 
Para maquillistas, esteticistas y dueñas de salones pequeños que trabajan solas o con poco personal y aún manejan sus citas manualmente.
 
---
 
## Qué hace
 
Una página web donde el cliente puede:
 
- Ver servicios y precios
- Elegir fecha y hora
- Enviar su reserva lista por WhatsApp
Sin necesidad de estar preguntando nada.
